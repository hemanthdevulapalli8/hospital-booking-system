# 🏥 Hospital Booking System

A full-stack hospital appointment booking system built with React, Node.js, Express, and MongoDB.

## Features

- 👤 Patient registration and login
- 🩺 Doctor listing and profiles
- 📅 Book, view and cancel appointments
- 👨‍⚕️ Doctor dashboard to view appointments
- 🔐 Admin dashboard to manage users and appointments
- ✅ Role-based access (Patient, Doctor, Admin)

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS, Axios

**Backend:** Node.js, Express, MongoDB, Mongoose, JWT
